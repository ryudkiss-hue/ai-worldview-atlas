## Troubleshooting

### "Platform not loading"
```bash
# Check if server is running:
npm start

# Check port 3000 is free:
lsof -i :3000

# If port in use, kill it:
kill -9 $(lsof -t -i :3000)

# Restart:
npm start
```

### "Responses not saving"
```bash
# Verify API endpoint is live:
curl http://localhost:3000/api/assessment/status

# Check data directory exists:
ls -la data/pilot-responses/

# Check server logs for errors
```

### "Prolific integration not working"
```bash
# Verify completion code format:
# Should be: ASSESS-XXXXXXXX-COMPLETE

# Test completion code manually:
# Prolific study settings → Completion URL
# Set to: http://yoursite.com/thank-you?completion=[%COMPLETION_CODE%]
```

### "Not getting responses"
```bash
# Check recruitment email reached lists
# Resend with updated copy if needed
# Extend recruitment to more channels
# Offer $6 instead of $5 if budget allows
```
